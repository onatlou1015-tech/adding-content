🏴‍☠️ Basic Elements ⚓ Straw Hat Crew · adding content

🏴‍☠️ LEARNING OUTCOME

  • Identify and differentiate between the HTML tags for hyperlinks (<a>), images (<img>), ordered lists (<ol>), and unordered lists (<ul>).
  • Use the semantic elements in html such as: <article>, <aside>, <details>, <figcaption>, <figure>, <footer>, <header>, <main>, <mark>, <nav>, <section>, <summary>, and <time>, and the non-semantic <div> and <span>
  • Construct a functional hyperlink to an external website using the href attribute.
  • Construct a functional hyperlink between two pages of their own website (internal link).
  • Embed an image into a web page using the src and alt attributes.
  • Create both ordered and unordered lists to organize information logically on a web page.
  • Synthesize these skills to create a cohesive "My Favorite Things" webpage that is properly linked into their existing project.

⚓ ESSENTIAL QUESTIONS

  1. How can we make our web pages more useful and interesting for visitors?
  2. Why is it important to organize information, and how can HTML help us do that?
  3. How do websites connect to each other, and how can we make our own website a "network" of pages?
